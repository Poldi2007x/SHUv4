Tutorial How to Use:

1. Download it and extract it
2. Start SHUV4.exe
3. Check if your computer is compatible. If it doesn't work, here are many reasons why:
- I'm lazy af: As mentioned, the code is messy and I probably didn't optimize it for your specific setup.
- Bluetooth Module: You probably don't have a Bluetooth module, or your drivers are from the stone age.
- Antivirus is paranoiac: Windows Defender or your Antivirus might delete files because they don't trust my unsigned .exe. (Just disable it or add an exclusion).
- Admin Rights: You didn't right-click and "Run as Administrator." It needs power!
- Missing Drivers: You might be missing the "Visual C++ Redistributable" or some .NET Framework stuff that I forgot to package.
- Path names: You extracted it to a folder with weird emojis or symbols in the name. Keep it simple (like C:\SHUV4).
